# GoSkilled — Public Experience (design charter, founder-directed 2026-07-11)

> This is NOT "marketing pages." It is the **Public Experience** — Brand · Trust · Sales · Conversion · Community · First Impression. The website is the **beginning of the complete GoSkilled experience**, not a set of isolated screens. Ground everything in `Experience_System_v1.0` · `DESIGN_DIRECTION.md` (constitution) · `ThreeState_RichHonestZero_Principle`.

## Objective
Every public page builds trust, communicates value, creates excitement, and **naturally guides the visitor toward registration** and becoming an active learner/affiliate. Design decisions serve those goals — not decoration.

## Think journey, not pages
Design ONE continuous journey; each page leads naturally to the next:
`Landing → Curiosity → Trust → Problem → Solution → Courses → Packages → Honest Proof → FAQ → WhatsApp → Register.`
**Emotional arc:** Curiosity → Trust → Excitement → Confidence → Action.

## First-impression rule (every landing surface)
In the first **5–10 seconds** the visitor must understand: **What is GoSkilled? · Why trust it? · Why is it different? · What do I do next?**

## Component system (reusable, on Experience-System tokens)
Hero · Feature Sections · Timeline · Comparison · Pricing · FAQ · CTA · **Honest** Testimonials · Trust layer · Benefits · Video · Stats · Community · Founder.

## Conversion rules (every page answers)
**Primary CTA · Secondary CTA · Trust layer · Next action.** e.g. Course page → Enroll → WhatsApp → Package → FAQ.

## Mobile (Tier-2/3)
Thumb-first · one-hand · scroll rhythm · **sticky CTA** · bottom actions · <2s first paint.

## Motion (device-tiered, each earns its place)
Hero reveal · card reveal · section transition · CTA pulse · (tasteful) image parallax · number animation · progress reveal. Full on capable devices; reduced/instant on low-tier + `prefers-reduced-motion`.

## Identity split
**Website = Emotion** (brand, trust, aspiration). **App = Execution** (productivity, clarity). Different feeling; shared tokens.

## Non-negotiable locks
Honest (D-29) — no fake testimonials/counts/stats; **"Founding Batch"** framing. **Rich-but-honest zero states** (ThreeState law). Re-skin in place (no logic change). WCAG AA. Shared Experience-System tokens (green/gold/charcoal · Sora/Inter). References for inspiration only — **Apple · Stripe · Linear · Notion · Framer · Duolingo · Coursera — never copy**; build a unique GoSkilled identity.

## Pre-code UX audit (every page, before building)
Would someone **remember** this tomorrow? · Would they **share** it? · Would they **trust** us? · Would they **register**? · Does it feel **premium**? · Do they understand GoSkilled in **10 seconds**? — if not an obvious YES, improve the design first.

## Scope
Homepage · Courses · Course Detail · Packages & Pricing · Webinar · About · Contact · FAQ · Blog · Videos. Branch `gps-marketing`; GATE — no merge; park + review packet.
